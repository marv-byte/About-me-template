# Simple "About me" template
A simple "About me" template made with HTML, CSS and JavaScript designed for self-host.
Perfect for developers or gamers who want a simple page.

## Features

* **100% self-hosted:** Just one file and no account on profile platforms needed to build your page.
* **Easy customization:** Clear comments show where you can add your info and change the colors.


## How to use
1. **Download the file**
2. **Customize it** Open it in an editor like Visual Studio Code or Notepad and follow these steps:
    * Change background color
    * Put in your name/gamertag, age, hobbies and other thing you want to add.
    * Add an image (recommended size 200px by 200px)
    * Paste your social media links into the JavaScript variables at the top
3. **Host it**
